## Event 1: FCJ 2025 Kickoff Event

**Event Name:** First Cloud Journey 2025 - Kickoff Event

**Date & Time:** Full Day, September 8, 2025

**Location:** Ho Chi Minh City, Vietnam

**Role:** Participating Intern

### Event Overview

This event brought together all selected interns, program coordinators, mentors, and AWS community leaders to launch the 2025 cohort. It was an intensive orientation and networking session designed to help interns understand program expectations, connect with peers, and present initial project ideas.

### Key Topics Covered

- **FCJ Program Structure**: Timeline, milestones, evaluation criteria, and expectations

- **Mentorship Framework**: How to work with mentors and coordinators throughout the program

### Sessions Attended

1. **Welcome and Program Overview** - FCJ Program Director

   - 2025 program structure and timeline
   - Success stories from previous cohorts
   - Evaluation criteria and graduation requirements

2. **Project Ideation Workshop** - Technical Mentors

   - Brainstorming project topics aligned with AWS services

   - Technical feasibility assessment

3. **Intern Project Presentations** - All Participating Interns

   - 5-minute pitch of initial project ideas
   - Q&A with mentors and fellow interns
   - Feedback on project scope and technical approach
   - Networking and finding potential collaborators

4. **AWS Community Introduction** - AWS User Group Leaders
   - Overview of AWS Vietnam community activities
   - Upcoming events and workshops
   - How to contribute to the community
   - Building professional networks in cloud computing

### Outcomes and Value Gained

- **Program Clarity**: Gained complete understanding of FCJ program structure, expectations, and success criteria
- **Project Validation**: Presented IoT Security Infrastructure project idea and received positive feedback from mentors
- **Peer Connections**: Met 25+ fellow interns from various universities, established collaboration channels
- **Mentor Network**: Connected with 5 technical mentors specializing in AWS security and IoT
- **Motivation Boost**: Inspired by success stories from previous FCJ cohorts who went on to AWS careers

### Impact on Internship Journey

This kickoff event set the foundation for my entire internship:

- Validated project topic choice and received guidance on scoping
- Established relationships with mentors who provided ongoing support
- Understood program milestones and planned accordingly
- Gained confidence from seeing the supportive FCJ community
