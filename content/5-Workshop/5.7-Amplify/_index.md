#### Overview
Connect your repo and deploy the frontend with AWS Amplify.

### Steps

1) Open **Amplify Console** → **New app** → **Host web app**  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify1.jpg)

2) **Connect repository & branch** (e.g., GitHub)  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify2.jpg)

3) **Authorize** and select the target **branch**  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify3.jpg)

4) Review **build settings** (amplify.yml) → keep defaults or adjust as needed  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify4.jpg)

5) Click **Save and deploy** to start the pipeline  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify5.jpg)

6) Wait for build + deploy to finish → status **Deployed**  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify6.jpg)

7) Open the **live URL** to verify the app  
   ![Amplify](/5-Workshop/5.7-Amplify-img/amplify7.jpg)
